Функция вывода printf языка C

Особенности языка

Функция printf() в языке Си предназначена для форматированного вывода. Она переводит данные в символьное представление и выводит полученные изображения символов на экран.

Общая форма записи функции printf(): printf("СтрокаФорматов", объект1, объект2, …, объектn);

Примеры верных строк из языка

В соответствии с разработанной автоматной грамматикой синтаксический анализатор (парсер) функции вывода printf языка C будет считать верными следующие записи:

printf("x = %f \n", r);
printf("x = %f \n", 12);
printf("Hello Word!");
